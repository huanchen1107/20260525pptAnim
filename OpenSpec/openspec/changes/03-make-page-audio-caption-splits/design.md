# Design вЂ“ Make Page Audio Splits & Slide Packages

## Overview
The goal of this change is to transform a single video (`A2Z-original.mp4`) and its Chinese subtitle file (`A2Zsrt.srt`) into a collection of perвЂ‘page assets that can be used for PPT animation.

### Asset Generation
1. **Audio Splitting** вЂ“ Using the subtitle timestamps, each nonвЂ‘empty subtitle block is cut from the extracted audio file (`A2Z-original-audio.mp3`). The resulting files are named `audioвЂ‘N.mp3` where `N` starts atвЂЇ1.
2. **Slide Extraction** вЂ“ Slide boundaries are detected via `ffmpeg` sceneвЂ‘change detection (`gt(scene,0.4)`). The first slide is forced to start at **00:01** to satisfy the user requirement.
3. **Packaging** вЂ“ For every page a folder `slideвЂ‘N/` is created containing:
   - `audioвЂ‘N.mp3`
   - `slideвЂ‘N.png` (image captured at the slide start timestamp)
   - `process_page.sh` вЂ“ a tiny helper script that can regenerate the assets for that page.

### Key Constraints
- **SlideвЂ‘1 start time** must be exactly 1вЂЇsecond.
- **Silent subtitles** (empty text) are skipped вЂ“ no folder/audio is created for them.
- All processing is performed with `ffmpeg -c copy` for audio (lossless) and PNG extraction for slides.
- The workflow is fully scriptвЂ‘driven; no manual editing of timestamps is required.

## Dependencies
- `ffmpeg` (required for both audio cutting and image extraction).
- `awk` (standard macOS utility) for parsing the SRT file).
- `whisper` (OpenAI Whisper CLI) for speechвЂ‘toвЂ‘text caption generation.

## Caption Generation
- After `audioвЂ‘N.mp3` files are created, run the `generate_captions.sh` script. It iterates over each `slideвЂ‘N/` folder, invokes `whisper` on the audio file, and writes the transcript to `captionвЂ‘slideвЂ‘N.txt` in the same folder.
- The script uses the `base` model for speed and stores the output as plain text.
- Captions are then available for downstream rendering steps (e.g., subtitle overlay in HTML).


## Failure Modes & Mitigations
- **Scene detection false positives/negatives** вЂ“ The `THRESHOLD` variable can be tuned. If slide boundaries are off, the user can adjust the threshold in `split_pages.sh`.
- **Missing subtitles** вЂ“ If a subtitle entry has no text, it is ignored, preventing empty audio files.
- **File path errors** вЂ“ All paths are relative to `user/assets`; the script aborts on any missing input file.
